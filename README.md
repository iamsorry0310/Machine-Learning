## Project Overview
This project aims to predict house prices using a variety of features such as location, size, and condition of the house. We use multiple machine learning models to identify the best-performing model for this task.

## Dataset
The dataset used for this project is the [Kaggle House Prices dataset](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data). It contains 1460 entries with 81 features.
